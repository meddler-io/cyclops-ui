# Application flow

## New Assessment
1. Choose App
2. We already know if it is android,ios,webapp, or webservice
3. Choose assessment type:
    1. webapp -> pentesting , static , dynamic , external-assessment
    2. ios -> pentesting , static , dynamic , external-assessment
    3. android -> pentesting , static , dynamic , external-assessment
    4. webservice -> pentesting , static , dynamic , external-assessment

4. Pentesting type:
    1. static
    2. dynamic
    3. pentest
    4. external-assessment
